**Estudo de Fundos Imobiliários**

Esse projeto utiliza NotebookLM como ferramenta de IA para auxiliar no estudo e conhecimento de fundos imobiliários (FIIs). O NotebookLM pode ser acessado pelo link abaixo

https://notebooklm.google.com/notebook/ea743ac2-9cd0-4e8e-8c42-1f4a10f8a121
----------
**OBJETIVO DO PROJETO**
  - Explicar para quais os tipos de FIIs
  - Auxiliar na tomada de decisão no investimento em FIIs
  - Gerar material didático das principais características de FIIs
  - Tirar dúvidas quanto a dividendos, valorização e desvalorização.
-----------
**OBSERVAÇÕES**

Este projeto tem finalidade **educacional e acadêmica** e não substitui aconselhamento financeiro profissional.
